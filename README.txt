# Job Application CRUD Project

## Instructions

1. Set up your database with the name `imjol_job`.
2. Create a table `jobs` using the SQL code provided in Step 2.
3. Configure your database connection in `config/database.php`.
4. Navigate to the `public` directory and access `index.php` to view the job applications.
5. Use `create.php` to add new job applications.
6. Use `update.php` to edit existing applications.
7. Use `delete.php` to remove an application.

For any issues, check that your database is set up correctly and PHP errors are enabled for debugging.


Step 1: Set Up Your Project Folder

1. Create a Project Folder
    Create a folder called office (or any name you prefer) where all your files will be stored.

2. Create Folders for Your Project Structure
    Inside your project folder, create the following structure:

Office/
├── config/
│   └── database.php
├── public/
│   ├── create.php
│   ├── update.php
│   └── delete.php
├── assets/
│   └── css/
│       └── style.css
├── index.php
└── README.txt
